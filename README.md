# 🛒 E-Commerce Web Application

## 📌 Overview

This is a fully functional **E-Commerce Web Application** developed using Angular. The application allows users to browse products, manage a shopping cart, and place orders. A mock backend is implemented using JSON Server to simulate real-world API interactions.

---

## 🚀 Features

* 🔐 User Authentication (Login / Signup)
* 🛍️ Product Listing & Search
* ➕ Add to Cart / Remove from Cart
* 📦 Order Placement
* 👨‍💼 Seller Module (Add / Manage Products)
* 🔄 Real-time data handling using REST APIs
* 📱 Responsive UI

---

## 🛠️ Technologies Used

* Angular 14
* TypeScript
* HTML5 & CSS3
* Bootstrap
* JSON Server (Mock Backend)
* REST API

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Shahida1111/angular-ecom-main.git
cd angular-ecom-main
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Run JSON Server (Backend)

```bash
json-server --watch db.json --port 3000
```

### 4️⃣ Run Angular App

```bash
ng serve
```

### 5️⃣ Open in browser

```
http://localhost:4200
```

---

## 🔗 API Endpoints (JSON Server)

* `http://localhost:3000/products`
* `http://localhost:3000/users`
* `http://localhost:3000/seller`
* `http://localhost:3000/cart`
* `http://localhost:3000/orders`
